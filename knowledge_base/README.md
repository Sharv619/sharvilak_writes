# Personal Blog Knowledge Base

This folder is a consolidated repository of all blog-related data, intended for use in a RAG (Retrieval-Augmented Generation) system or chatbot.

## Contents
- `himanshoe.WordPress.2026-04-30.xml`: The complete WordPress export.
- `export.zip`: (Pending) The official Instagram data export.
- `scraped_posts/`: Media and metadata for public Instagram posts.
- `wordpress_ig_context.json`: Extracted snippets linking WordPress posts to Instagram content.
- `scripts/`: Various Python scripts for data extraction and processing.
